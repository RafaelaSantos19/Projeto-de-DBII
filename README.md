
# AcessiTransporte

Projeto completo (Front-end + Back-end) desenvolvido por Rafaela Santos da Costa como Trabalho de Conclusão de Curso.

## Descrição
Sistema web de transporte acessível para pessoas com deficiência, permitindo simulação de corridas, cálculo de preço e visualização de informações.

## Tecnologias
- HTML5, CSS3, JavaScript
- PHP
- SQLite

## Como rodar
1. Coloque a pasta AcessiTransporte_FULL dentro do htdocs
2. Inicie o Apache
3. Execute http://localhost/AcessiTransporte_FULL/api/database/setup.php
4. Acesse http://localhost/AcessiTransporte_FULL/index.html

## Rotas da API
- GET /api/rides
- POST /api/rides
